# PneumoApp_v6_80_R176_WINDOWS_CLEAN_R23_2026-03-20

Основные изменения R23:
- 3D Animator теперь выводит положение колёс по explicit hardpoints верхнего/нижнего рычага, а не по замороженному wheel_center.
- Визуализация контакта колеса с дорогой переведена с прямоугольного пятна на эллиптическое пятно в плоскости дороги.
- Поршни цилиндров сделаны заметнее: увеличена толщина/радиус диска и добавлены маркеры центров поршней.
- Дорога стала светлее и получила плотные продольные/поперечные направляющие линии без прежнего замедления от 900 узлов.
- START_PNEUMO_APP.cmd больше не держит лишнее чёрное окно на pause; Desktop Animator запускается без transient console windows.

Честное ограничение:
- Полноценные toe/camber/track изменения теперь визуализируются по hardpoints, но если сама текущая геометрия остаётся симметричной/параллелограммной, toe/camber могут оставаться близкими к нулю. Новые углы не выдумываются в animator.
