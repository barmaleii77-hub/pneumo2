# Interfeis772 (база: UnifiedPneumoApp_UNIFIED_v6_22_WINSAFE)

Цель релиза: сделать UI **понятным человеку** и убрать основные UX-боли:
- нет горизонтального скролла в параметрах/тестах,
- меньше ручного ввода «в таблицу» — больше карточек, чекбоксов, ручек,
- 6 смысловых разделов + «Рабочее место»,
- русификация (где уместно) + подсказки (`help=`) и пояснения разделов.

## Что сделано

### 1) Навигация приложения (app.py)
- Группировка страниц по смыслу: **Основное / Проектирование / Анализ / Оптимизация / Инструменты / Сервис**
- Русские названия пунктов меню
- `st.navigation(..., position="top")`

### 2) Пояснения разделов и tooltips
- В симуляторе добавлены:
  - тумблеры **«Показывать пояснения»** и **«Расширенный режим»**
  - развёрнутые блоки «ℹ️ что здесь делать» для разделов
- Используется `help=` у виджетов Streamlit (официальный tooltip).

### 3) Разделы симулятора (pneumo_ui_app.py)
- Добавлен переключатель режима:
  - **Рабочее место**: запуск + результаты
  - **Полный интерфейс**: разделы **Модель / Параметры / Тесты / Прогон / Результаты / Инструменты**
- Тяжёлые блоки UI скрываются, когда не выбран соответствующий раздел.

### 4) Параметры: «список + карточка» вместо широкой таблицы
- Редактор параметров теперь:
  - фильтр по группе / поиск / «только оптимизируемые»
  - компактный список (preview)
  - карточка выбранного параметра: значение + чекбокс оптимизации + диапазон (range slider)

### 5) Тест‑набор: «список + карточка»
- Компактный список тестов (вкл/имя/тип/dt/t_end)
- Карточка выбранного теста:
  - базовые поля (вкл/имя/тип/dt/t_end/t_step)
  - параметры сценария (ay/ax/A/f/…)
  - цели/лимиты (`target_*/settle_*`) спрятаны в «Расширенно»

### 6) Русификация страниц
- Переведены заголовки страниц и добавлены блоки «ℹ️ Что здесь делать».

## Где что лежит

- `app.py` — вход, мультистраничная навигация.
- `pneumo_solver_ui/pneumo_ui_app.py` — основной симулятор UI.
- `pneumo_solver_ui/pages/*.py` — страницы приложения.
- `docs/UX_SOURCES_RU.md` — источники по UX решениям.
- `docs/context_packs/` — контекст проекта (AI_PROMPTS_PACK / PROJECT_PRESERVE_PACK).

```bash
pip install -r requirements.txt

```
