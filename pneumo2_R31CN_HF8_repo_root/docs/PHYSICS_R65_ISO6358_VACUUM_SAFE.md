# PHYSICS R65: ISO 6358 vacuum-safe phi() and unified ANR (ISO 8778)

Этот документ описывает физическую мотивацию и конкретные правки релиза **FizikaR65**.

## 1) Контекст: ISO 6358 и функция phi(pr)

В ISO 6358 паспортные параметры расходного элемента:
- **C** — sonic conductance,
- **b** — critical pressure ratio,
- **m** — shape parameter (для субзвуковой части),
- иногда dPcrack — «cracking pressure» (для обратных клапанов и т.п.).

В расчётах используется безразмерная функция **phi(pr)**, где
`pr = p_down / p_up`.

Физически важное свойство:
- при `pr -> 0` поток переходит в режим **choked**, и зависимость от downstream давления
  прекращается (phi стремится к 1),
- при `pr -> 1` перепад давления исчезает, и расход должен стремиться к нулю (phi -> 0).

## 2) Что было неправильно

В двух v9-моделях функция `iso6358_phi(pr,...)` возвращала **0** при `pr <= 0`.
Это приводило к «искусственному запиранию» расхода при вакууме/нулевом pr.

## 3) Что сделано в FizikaR65

1) Исправлено: `pr<=0 => phi=1`, `pr>=1 => phi=0`.
2) Введён единый модуль `pneumo_solver_ui/iso6358_core.py`:
   - ANR по ISO 8778: 100 kPa, 20°C, RH=65%,
   - расчёт плотности влажного воздуха (идеальная смесь) + насыщ. давление по Buck,
   - эталонная реализация phi(pr).
3) `iso6358_fit.py` отвязан от тяжёлой v8-модели и использует `iso6358_core`.
4) `iso6358_system.py` унифицирован по константам/phi.
5) Добавлены unit-тесты, фиксирующие ключевой физический контракт.
