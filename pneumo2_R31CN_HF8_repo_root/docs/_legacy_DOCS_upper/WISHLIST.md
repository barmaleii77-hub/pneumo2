# WISHLIST (идеи)

## UX/GUI
1) **Режим «анти‑идиот»** (по умолчанию включён):
   - блокировать кнопки действий, пока не выполнены обязательные шаги;
   - показывать «что не готово» прямо на кнопке;
   - подтверждение для разрушительных действий.

2) **Контекстные подсказки‑баннеры**:
   - короткая строка (1‑2 предложения) над каждым блоком,
   - автоматическое скрытие, когда пользователь сделал нужное действие.

3) **Быстрые пресеты** («Грунтовка/Город/Магистраль», «Разгон/Торможение/Перестроение»):
   - один клик → заполнение значений → сразу запуск.

4) **Единый центр экспорта**:
   - CSV/NPZ/JSON/SVG/MP4 → всё в одном месте,
   - «экспортировать текущую сессию».

## Анимация/визуализация
1) **Веб‑превью 3D** (упрощённое):
   - WebGL (three.js) компонент для показа рамы (параллелепипед) и колёс (цилиндры),
   - визуализация дороги (профиль + поворот),
   - скорость дороги соответствует расчётной.

2) **Анимированная теплокарта**:
   - вертикаль: узлы/контуры,
   - горизонталь: время,
   - цвет: давление/расход,
   - playhead синхронизирован с анимацией.

3) **Дублирование текстом**:
   - «поворот влево/вправо, радиус, скорость, тип участка»
   - «манёвр: разгон/торможение/перестроение»

## Надёжность
1) **Автотесты UI**:
   - smoke‑тест запуска streamlit,
   - прогон `ui_selfcheck` в CI,
   - тест «профиль сохранился/восстановился».

2) **Логирование пользовательских действий**:
   - действия по кнопкам,
   - смена вкладок,
   - загрузка/выгрузка файлов,
   - экспорт.
