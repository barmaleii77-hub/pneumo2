# Ограничения и неоткрытые окна — V10

- В этом проходе не открывались живые внутренности окон GUI диагностики, GUI отправки результатов, отдельного Compare Viewer, Desktop Mnemo и Центра desktop-инструментов.
- Выводы опираются на подтверждённые launchpoints, на предыдущие report-only слои и на проектный канон, а не на полный runtime walkthrough каждого отдельного окна.
- Этот шаг не является кодовой рекомендацией и не содержит правок проекта — только report-only выводы и предложение по иерархии.
