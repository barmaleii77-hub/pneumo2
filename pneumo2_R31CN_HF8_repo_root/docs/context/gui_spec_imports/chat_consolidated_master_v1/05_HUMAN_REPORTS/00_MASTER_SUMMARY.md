# Human reports — сводка

В report-only ветку включены только финальные слои без ранних дублей:

- `v10` — launcher-shell hierarchy
- `v11` — current vs canonical pass по launchpoint-only окнам
- `v12` — internal routes
- `v13` — canonical window operations
- `v14` — tree / dock / context
- `v15` — state continuity / repair loops
- `v16` — visibility priority / must-see states

Эта ветка отвечает на вопросы:
- что пользователь видит и чего не понимает;
- что хорошо, что плохо и как поправить;
- что должно быть top-level, а что secondary;
- какие состояния обязаны быть видимы сразу.
