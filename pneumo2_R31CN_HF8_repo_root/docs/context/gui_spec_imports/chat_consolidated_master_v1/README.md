# Сводный архив наработок чата

Этот архив собирает ключевые результаты чата **без повторений и избыточности**.

## Как устроен архив
- `01_SOURCE_CONTEXT` — канонические prompt/контекст проекта и исследовательская база.
- `02_CODEX_SPEC_FINAL` — финальная сводная спецификация для CODEX, актуализированная с учётом launcher-shell анализа.
- `03_REPO_AUDIT` — аудит локального/удалённого repo и соответствия базе знаний.
- `04_GRAPH_ANALYSIS` — итоговая графовая ветка: reconciliation, workspace-subgraph-ы, cost/entropy.
- `05_HUMAN_REPORTS` — report-only ветка глазами пользователя: launcher, окна, state continuity, visibility.

## Что сознательно НЕ включено
Не включены десятки промежуточных архивов и версии, которые полностью или частично заменены более поздними слоями. Их список и причина исключения есть в `06_INDEX/SUPERSEDED_AND_EXCLUDED.csv`.
