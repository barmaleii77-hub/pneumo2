# Логика свёртки без повторений

## Что сохранено
Сохранены только:
1. Финальный сводный CODEX-spec слой.
2. Финальный аудит repo.
3. Финальные graph-слои, которые действительно дополняют друг друга.
4. Финальные report-only пользовательские слои.

## Что убрано
Убраны:
- промежуточные spec-версии, если их материал уже встроен в финальный сводный слой;
- implementation-ветка (v7-v11 GUI code scaffolding), потому что пользователь просил не код, а анализ/спецификацию;
- ранние human/simulation отчёты, которые заменены более поздними report-only слоями;
- ранние графовые версии, если их функции уже покрыты v17-v21.

## Почему это не потеря
Для каждого исключённого артефакта сохранена запись в `SUPERSEDED_AND_EXCLUDED.csv` с причиной исключения.
