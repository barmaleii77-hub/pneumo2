# Что плохо — V10

- Стартовый shell до сих пор ведёт себя скорее как каталог входов, чем как один сильный пользовательский маршрут.
- Редактор кольца ещё не воспринимается как доминирующий сценарный центр на первом экране.
- Диагностика и отправка результатов выглядят как два конкурирующих направления, а не как этапы одной операции.
- Встроенное сравнение и Compare Viewer дублируют друг друга без ясной иерархии.
- Редактор исходных данных и Центр тестов недостаточно явно показаны как обязательные шаги до baseline и optimization.
- Desktop Mnemo и Центр desktop-инструментов видны слишком рано для обычного пользователя.
