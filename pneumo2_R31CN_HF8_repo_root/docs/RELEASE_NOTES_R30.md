# Release Notes — Обобщение R30 (2026‑01‑23)

## Критические фиксы

### 1) Исправлен краш старта/детального прогона: `NameError: t_sec is not defined`
- В блоке **детального прогона (full‑лог)** метаданные сохранялись с полем `t_sec`, но переменная не была определена.
- Теперь `t_sec` = **время выполнения детального прогона (wall‑time)**, измеряется через `time.perf_counter()`.

### 2) Исправлен потенциальный «бесконечный перерасчёт» детального прогона
- В предыдущем коде была **сломана структура try/except** (двойной `except Exception`, `detail_end` логировался только при ошибке сохранения кэша).
- Переписано на понятный шаблон:
  - `detail_start` → запуск симуляции → `detail_cache_save_error` (если нужно) → `detail_end`
  - при ошибке симуляции/парсинга: `detail_error`
  - `finally`: **гарантированный сброс `in_progress`** и установка `last_end_ts`.

### 3) Авто‑детальный прогон больше не будет «молотить» бесконечно после ошибки
- Добавлен флаг `failed_key/failed_err` в `detail_guard`.
- Если предыдущая попытка авто‑прогона для того же `cache_key` завершилась ошибкой — авто‑запуск **подавляется** (показывается предупреждение).
- Ручная кнопка «Рассчитать полный лог…» всегда позволяет повторить.

## UX/производительность

- 3D режим: **траектория по умолчанию = «Статика (без движения)»** (чтобы не создавалось ощущение, что продольная/поперечная динамика «дорисовывается» визуализацией).

### Авто‑пауза Play при переключении режима отображения
- При смене режима (Графики / Пневмо / Анимация / Мех 3D) отправляется `playhead_cmd: set_playing=False`.
- Это защищает от ситуации, когда в `localStorage` остался `playing=true` и при входе на анимацию GPU сразу улетает в 30 FPS.

## Логи
- В `logs/ui_combined.log` теперь корректно пишутся:
  - `detail_start` (параметры запуска)
  - `detail_end` (elapsed_s, rows)
  - `detail_error`
  - `detail_autorun_suppressed_after_error`

## Что НЕ делалось в этом релизе
- Не менялась физика матмодели.
- Не «подрисовывалась» анимация синтетикой: крен/тангаж/дорога берутся из расчётного `df_main` (если столбцов нет — рисуются нули, без «фейков»).
