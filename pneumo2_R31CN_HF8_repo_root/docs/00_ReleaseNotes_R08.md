# Обобщение — релиз R08

Дата: 2026-01-20

## Главные изменения (по логам/ошибкам из запуска на Windows)

### 1) Исправлена загрузка Streamlit-компонентов (mech_anim / mech_car3d / playhead_ctrl / pneumo_svg_flow)
**Симптом (из ваших логов/сообщений):**
- `Your app is having trouble loading the pneumo_ui_app.mech_anim component`
- `Error: Unrecognized component API version: 'undefined'`
- 2D/3D анимации показывали только «пустые панели» (канвасы без отрисовки).

**Причина (практически):**
- для “no-tooling” компонентов Streamlit критично, чтобы сообщение `streamlit:componentReady` приходило *раньше*, чем произойдут любые JS-ошибки/тяжёлые операции;
- в некоторых версиях Streamlit наблюдается «капризность» к полю `apiVersion` (и/или чтение `args.apiVersion`).

**Что сделано в R08:**
- во все HTML-компоненты добавлен **ранний и защищённый (guarded) handshake**:
  - `componentReady` отправляется **сразу после объявления функции отправки сообщений**, до `init()` и до любой тяжёлой логики;
  - отправляется сразу в двух вариантах: `apiVersion` на верхнем уровне и `args.apiVersion` (на случай различий протокола).
- вызовы `init()` обёрнуты в `try/except` (JS try/catch), чтобы одиночная ошибка не “ломала” handshake полностью.

Файлы:
- `pneumo_solver_ui/components/mech_anim/index.html`
- `pneumo_solver_ui/components/mech_car3d/index.html`
- `pneumo_solver_ui/components/playhead_ctrl/index.html`
- `pneumo_solver_ui/components/pneumo_svg_flow/index.html`

> Если на вашем Streamlit/браузере всё равно появится ошибка компонента — используйте “Встроенный (matplotlib)” движок анимации. Он независим от компонентов.

### 2) “Режимы (string)” в UI теперь в 2 колонки (не растягивается на всю ширину)
**Симптом:**
- селекторы/выпадающие списки занимали всю ширину экрана, хотя значения короткие (`constant` и т.п.).

**Что сделано:**
- блок “Режимы (string)” теперь раскладывается в **2 колонки**, поэтому селекторы визуально компактнее.
Файл: `pneumo_solver_ui/pneumo_ui_app.py`

### 3) Улучшено сообщение про отсутствие Plotly
**Симптом:**
- “Plotly не установлен — Graph Studio недоступен.”

**Что сделано:**
- сообщение стало более практичным: указывает на `requirements.txt` и на one-click bat.
