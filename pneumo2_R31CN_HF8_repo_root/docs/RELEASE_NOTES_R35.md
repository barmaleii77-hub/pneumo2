# RELEASE NOTES — Obobshchenie R35

Дата: 2026-01-23

## Главное
Этот релиз — **точечный фикс краша UI** в блоке таймлайна (Playhead) на странице графиков/анимации.

## Исправления
1) **Краш `NameError: events is not defined`**
- Исправлено место, где в аргументы playhead‑компонента передавалась переменная `events`, которой в этой области видимости нет.
- Теперь используется корректная переменная `events_list`, вычисленная через `compute_events()`.

2) **Корректное время событий**
- В `compute_events()` время события хранится в ключе `t`, а в UI ранее читали `t_s`.
- Теперь время берётся как `ev.get('t', ev.get('t_s', 0.0))`, чтобы маркеры таймлайна не попадали в `t=0`.

## Что НЕ решено (важно)
- “Play дёргает ползунок / бесконечный пересчёт / GPU растёт при открытии вкладки Анимация” — это отдельная проблема связки **rerun‑петель** и **server‑sync**. В этом релизе чинится именно краш `events`.
