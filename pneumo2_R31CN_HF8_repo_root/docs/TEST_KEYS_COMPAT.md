# УСТАРЕЛО

Этот документ оставлен только как историческая заметка.

**Совместимость ключей / алиасы больше не поддерживаются.**

Актуальные правила и единый список канонических ключей:
- `00_READ_FIRST__ABSOLUTE_LAW.md`
- `01_PARAMETER_REGISTRY.md`
- `pneumo_solver_ui/contracts/param_registry.yaml`

Если в данных встретились legacy‑ключи (например `speed_mps`, `road_profile_csv` и т.п.) —
это считается ошибкой источника данных. Приложение должно **логировать предупреждение**
и требовать явного исправления в producer/suite, без авто‑миграции.
