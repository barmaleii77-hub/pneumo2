# RELEASE NOTES — R13 — 2026-03-18

## Что сделано
- Добавлена честная диагностика геометрии подвески в Desktop Animator.
- HUD теперь показывает краткий статус по рычагам и цилиндрам без выдуманной геометрии.
- selfcheck теперь отдельно предупреждает, если:
  - на угол сериализована только одна геометрия рычага при схеме double wishbone;
  - каналы цилиндров C1/C2 существуют, но геометрически совпадают по одной оси.
- Добавлен CLI-инструмент `python -m pneumo_solver_ui.tools.inspect_suspension_geometry <anim_latest.npz>`.

## Что показал разбор реального bundle
- В доступных локально bundle `654f1740-...zip` и `SEND_20260317_223459_auto-exit_bundle.zip` на каждом углу есть только **одна** геометрия рычага (`arm_pivot -> arm_joint`).
- Каналы цилиндров `C1` и `C2` присутствуют, но их верхние и нижние solver-points совпадают, то есть геометрически это **одна и та же ось**, а не два различимых цилиндра.

## Ограничение
Это исправление делает проблему **наблюдаемой и честно диагностируемой**, но не выдумывает второй рычаг и не раздвигает цилиндры искусственно. Для реальной визуализации двух рычагов и двух разных цилиндров на угол solver/export должен сериализовать отдельные точки для второй геометрии.
