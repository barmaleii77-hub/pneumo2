# Release Notes — R22

Дата: 2026‑01‑22

## Главное

### 1) 3D визуализация: «не видно / будто за пределами канвы»
Сделано:
- Добавлена кнопка **Reset view** рядом с настройками 3D (UI).
- Добавлена опция **DEBUG overlay** — рисует служебную строку `dataset/idx/t` прямо на канве, чтобы понять, что сцена реально обновляется.
- В компонент `mech_car3d` добавлен авто‑сброс камеры/панорамирования при смене `dataset_id`, чтобы после переключения теста/кэша машина снова попадала в кадр.
- Добавлена поддержка команд `cmd.reset_view` и `cmd.clear_selection` из Python → компонент.

### 2) Калибровка/Autopilot: согласование путей
Сделано:
- Страница Calibration (NPZ/CSV) теперь по умолчанию использует:
  - `workspace/osc/` как `osc_dir`
  - `workspace/calibration_runs/` как папку результатов
- Это согласовано с тем, куда симулятор экспортирует NPZ (`Export NPZ ...`).

### 3) Документация
Обновлено/добавлено:
- `docs/03_Troubleshooting.md` (R22)
- `docs/05_WhatIs_NPZ.md` (полностью переписан, с примерами)
- `docs/06_CalibrationAndAutopilot.md` (объяснение «зачем оба», и почему не всегда auto)
- `docs/11_TODO.md` — добавлены новые TODO по анимации/булевым флагам/one‑click pipeline.
