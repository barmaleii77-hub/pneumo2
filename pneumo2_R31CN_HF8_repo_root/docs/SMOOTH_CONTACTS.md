# Гладкий контакт шины (6.53)

## Что было
В fully_smooth_mode использовалась схема:
- `g_contact = smooth_pos_grad(pen)`
- `pen_in = pen * g_contact`

Эта схема действительно убирает паразитный оффсет в нуле, но для небольших отрицательных `pen` даёт `pen_in < 0`,
а значит и **F_tire < 0** (шина "тянет" дорогу).

## Что стало
Добавлены функции `smooth_pos0` и `smooth_pos0_grad`.

Использование:
- `pen_pos = smooth_pos0(pen, eps_pos)`  (>=0, и ровно 0 при pen<=0)
- `g_contact = smooth_pos0_grad(pen, eps_pos)` (0..1)
- `F_tire = k*pen_pos + c*smooth_pos0(pen_dot, eps_vel)*g_contact`

Так обеспечивается:
- отсутствие отрицательных сил,
- отсутствие паразитного оффсета в нуле,
- плавное включение контакта.

## Гладкость
Функция C1‑непрерывна (производная непрерывна), но вторая производная может иметь разрыв в 0 и eps.
Для большинства задач оптимизации этого достаточно; при необходимости можно сделать C2/C∞ вариант.
