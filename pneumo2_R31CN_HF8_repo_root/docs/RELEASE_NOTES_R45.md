# RELEASE_NOTES_R45

База: **ObobshchenieR43**

## Ключевой смысл релиза

R45 делает модель пневмосети заметно ближе к "железу" за счёт:

1) **ISO 6358 для регуляторов и клапанов сброса** — рёбра `reg_after` и `relief` перестали быть "просто отверстиями".
2) **Паспорт MC/VMR/2905** заполнен (без null), чтобы режим `использовать_паспорт_компонентов=True` работал корректно.
3) **Настраиваемые сопротивления** на ветках "инерционного тормоза" (N_* → магистраль) и зарядки аккумулятора (Р3 → accumulator).
4) Добавлен **Design Advisor** — авто-поиск узких мест: большие Δp, большие потери энергии, недостаточный запас по C.

## Изменённые файлы

См. `CHANGED_FILES_R45.txt`.

## Изменения по сути

### 1) ISO 6358 для `reg_after` и `relief`

- Если `модель_пассивного_расхода = iso6358`, то:
  - `reg_after` использует `C_iso,b,m` из рёбра (или оценку из площади), с учётом `reg_leak` и сглаживания по `dp_crack`.
  - `relief` использует `C_iso,b,m`, с учётом `relief_leak` и логики открытия от `p_set`.

### 2) Паспорт MC/VMR/2905

- В `component_passport.json` добавлены оценки Qn и ISO-параметров для:
  - MC104-R00 (Series MC): Qn≈1450 Nl/min
  - VMR 1/8-B10: Qn≈800 Nl/min
  - 2905: Qn≈700 Nl/min

Все эти значения помечены `_status: estimated`.

### 3) Новые параметры

- `коэф_прохода_сбор_в_магистраль` (default 1.0)
- `коэф_прохода_заряд_аккумулятора` (default 1.0)
- `начальное_давление_Ресивер1` (default 101325 Pa)
- `начальное_давление_Ресивер2` (default = давление_Pmin_питание_Ресивер2)
- `начальное_давление_Ресивер3` (default = давление_Pmin_питание_Ресивер2)

### 4) Design Advisor

- `pneumo_solver_ui/design_advisor.py` — CLI/модуль анализа.
- `pneumo_solver_ui/pages/03_Design_Advisor.py` — страница Streamlit.
