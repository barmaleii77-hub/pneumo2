# RELEASE NOTES — R42

Дата: 2026‑01‑24

## Главные изменения

### 1) Фикс падения анимации (fallback)
- Исправлена критическая ошибка `NameError: target_fps is not defined` в `pneumo_solver_ui/mech_anim_fallback.py`.
- Причина: в расчёте backlog использовалась переменная `target_fps`, которая отсутствовала в текущей версии функции.
- Теперь используется корректный `fps` из UI.

### 2) Стабилизация поведения Play в fallback-анимации
- В `_playhead_idx_control()` убрана агрессивная логика "догоняния" (`backlog`) по wall‑clock времени.
  Она давала скачки индекса/ползунка при рывках FPS/нагрузке и выглядела как "мечется".
- Воспроизведение теперь инкрементирует индекс предсказуемо: `idx += round(speed)` на тик UI, с защитным clamp.

### 3) Актуализация контекста требований (v2)
