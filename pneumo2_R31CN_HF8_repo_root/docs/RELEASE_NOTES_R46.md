# Release Notes — R46

Дата: 2026-01-25

## Ключевое

### 1) Раздельный выхлоп Р3: SCO → 2905 → ATM

- Добавлен переключатель `разделить_SCO_и_глушитель_2905` (по умолчанию **true**).
- При включении:
  - `дроссель_выхлоп_*` моделируется как SCO (регулируемый элемент, `C_min/C_max` + `alpha`).
  - `глушитель_выхлоп_*` моделируется как отдельный SIL‑элемент (ISO 6358 `C,b,m`).
- Добавлен малый мёртвый объём между SCO и 2905:
  - `объём_узла_между_дросселем_и_глушителем` (по умолчанию `5e-6 м³`).

## Паспорт

- В `pneumo_solver_ui/component_passport.json` добавлены позиции:
  - `2905 1/8`
  - `2905 1/4`
- Заполнены оценочные `Qn_Nl_min_at_6bar_to_atm` и ISO6358 (`C,b,m,Δpc`).

## Design Advisor

- Добавлены колонки:
  - `C_min_m3_s_Pa`, `C_max_m3_s_Pa`
  - `margin_C_max`
  - `sizing_hint`
  - `suggested_replacements` (список кандидатов замены по паспорту)
- Различаются случаи:
  - «настройка душит» (C_max достаточен)
  - «железо душит» (C_max недостаточен → нужен больший типоразмер)

## Параметры по умолчанию (default_base.json)

Добавлены:
- `разделить_SCO_и_глушитель_2905: true`
- `код_глушителя_выхлоп_Pmax: "2905 1/4"`
- `код_глушителя_выхлоп_Pmid: "2905 1/8"`
- `код_глушителя_выхлоп_Pmin: "2905 1/8"`
- `объём_узла_между_дросселем_и_глушителем: 5e-6`

## Совместимость

- Если нужно вернуть старое поведение (комбо `SCO+2905` как один элемент), установите:
  - `разделить_SCO_и_глушитель_2905 = false`
