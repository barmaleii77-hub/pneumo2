# Release notes — ObobshchenieR23

Дата: 2026-01-22

## Что исправлено

### 1) 3D анимация (mech_car3d) больше не «пустая»
В предыдущей сборке 3D мог не отображаться из-за JS-ошибки (использовалась переменная `isHover` вне области видимости при включённом `hover_contact_marker`).

В R23:
- исправлено вычисление hover-состояния для маркера контакта колеса с дорогой;
- добавлен защитный `try/catch` вокруг рендера — чтобы единичная ошибка не «убивала» весь кадр;
- добавлено ограничение FPS (≈30 FPS в режиме Play и ≈12 FPS в паузе), чтобы снизить загрузку CPU.

### 2) self_check.py больше не падает на Windows из-за символа “Δ”
Символ “Δ” в выводе консоли мог приводить к `UnicodeEncodeError: 'charmap' codec can't encode character`
в терминалах/кодировках, где этот символ не поддерживается.

В R23 заменены сообщения на ASCII-варианты (`dPcrack`, `dP`).

### 3) Меньше предупреждений Streamlit про use_container_width
Обновлён wrapper `safe_dataframe()`:
- сначала пытается `width="stretch"`,
- затем пробует вариант без `hide_index`,
- и только потом падает обратно на `use_container_width=True`.

## Что добавлено

### GUI запуск полной диагностики без консоли
Добавлено:
- `pneumo_solver_ui/tools/run_full_diagnostics_gui.py` (tkinter)
- `RUN_FULL_DIAGNOSTICS_GUI_WINDOWS.bat`

Это позволяет выбирать уровень диагностики (minimal/standard/full) и опции без ручного ввода команд.
