# Release notes — Obobshchenie R07

## Главные изменения

1. **Анимации 2D/3D стали "неубиваемыми"**
   - По умолчанию используется встроенная отрисовка на **matplotlib** (без Streamlit Components).
   - Компонентный режим (SVG/Canvas) оставлен как **экспериментальный** и включается вручную в UI (вкладка *Анимация → Механика*).

2. **Удобство запуска на Windows**
   - Добавлены:
     - `INSTALL_WINDOWS.bat` — создаёт `.venv`, ставит зависимости;
     - `RUN_WINDOWS.bat` — запускает Streamlit UI;
     - `RUN_ONECLICK_WINDOWS.bat` — если `.venv` нет, сначала ставит зависимости, затем запускает.

3. **UI: компактный макет**
   - В sidebar добавлен переключатель *“Сжатый макет”* — ограничивает ширину контента, чтобы короткие селекторы/списки не растягивались на весь экран.

## Почему это сделано

- Ты прислал ошибки:
  - *Plotly не установлен* → интерактивные графики выключаются.
  - *Unrecognized component API version: 'undefined'* → Streamlit custom component не инициализируется в браузере.

В R07 основной сценарий не зависит от кастомных компонентов: всё важное (минимум для оценки модели) можно увидеть без них.
