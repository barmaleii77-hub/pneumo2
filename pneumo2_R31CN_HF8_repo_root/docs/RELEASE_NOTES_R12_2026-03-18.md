# RELEASE NOTES — R12 — 2026-03-18

## Что исправлено

### 3D геометрия
- Исправлен ложный нулевой клиренс: box-рама больше не центрируется прямо по `frame_corner_*`.
- Зелёная стрелка скорости переведена в локальную плоскость рамы.
- В 3D включены честные solver-point линии рычагов и цилиндров.
- Добавлена подсветка пятна контакта как отдельный service-only mesh на поверхности дороги.

### Телеметрия / layout
- Телеметрия разнесена на отдельные dock-окна:
  - heatmap
  - quick corners
  - road profile
  - corner table
  - pressures
  - flows
  - valves
- Компактная сводка и визуальные переключатели остаются в отдельном компактном dock.

### 2D / масштаб
- Для фронтальных/боковых видов добавлен zoom колёсиком мыши.
- Двойной клик возвращает auto-scale.
- Road HUD держит auto-fit только пока пользователь сам не начал ручной zoom.

### 2D дорога
- Side-view использует реальный профиль дороги через `ensure_road_profile(...)`, если он доступен.
