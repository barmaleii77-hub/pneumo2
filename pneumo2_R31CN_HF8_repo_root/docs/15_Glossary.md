# Глоссарий

## NPZ
`*.npz` — это формат NumPy: ZIP‑архив, внутри которого лежат несколько массивов `*.npy` (включая имена).
Обычно используется для хранения синхронизированных временных рядов/сигналов без потери точности
и без проблем с локалью/разделителем дробной части.

## Baseline
Быстрый расчёт по выбранному тестовому набору (suite) для:
- первичной проверки матмодели,
- построения базовых графиков,
- выбора теста/кандидата для более тяжёлого «детального прогона».

## Детальный прогон (full log)
Расчёт с сохранением расширенного набора временных рядов по всем узлам (кинематика/динамика/пневматика),
включая события и каналы для анимации/диагностики.

## Калибровка (Calibration)
Подбор параметров матмодели под **измеренные данные** (NPZ/CSV), чтобы приблизить симуляцию к реальности.

## Autopilot
Автоматический конвейер, который запускает:
1) подготовку/проверку данных (NPZ/CSV),
2) калибровку/поиск параметров по заданной стратегии,
3) сравнение кандидатов по KPI,
4) (опционально) продвижение лучшего кандидата в baseline/«текущие параметры».
