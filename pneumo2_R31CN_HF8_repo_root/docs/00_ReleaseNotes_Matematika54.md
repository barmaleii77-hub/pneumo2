# Matematika54 — Release Notes

Дата: 2026-01-25  
База: ObobshchenieR43  
Предыдущий релиз: Matematika53

## Коротко

В этом релизе исправлены два критических дефекта механики, которые напрямую влияют на контакт «шина–дорога» и на отбойники:

1) **Контакт шины с дорогой теперь использует правильную относительную скорость**:  
`pen_dot = z_road_dot - z_wheel_dot` (а не `-z_wheel_dot`).  
Это критично для демпфирования шины и для корректного энерго‑баланса при «движущейся базе» (дорога).

2) **Отбойники теперь используют кинематическую скорость относительного перемещения**:  
`delta_dot = z_wheel_dot - z_body_dot`, а не `-s_dot` из функции объёмов.  
`s_dot` в ряде режимов принудительно «замораживался» на упоре (freeze), что делало демпфирование отбойников некорректным.

Дополнительно расширен smooth‑модуль: добавлена **аналитическая `road_func_dot(t)`**, добавлены колонки скорости/ускорения дороги и внедрён **механический энерго‑аудит** (в том же стиле, что и в основном «energy_audit» модуле).

## Что сделано

### 1) Правильная кинематика контакта «шина–дорога» (base excitation)

**Было:**  
`pen_dot = -zw_dot`  
Т.е. дорога считалась неподвижной.

**Стало:**  
`pen_dot = z_road_dot - zw_dot`  
Демпфер шины реагирует на **относительную скорость** (колесо относительно дороги).

Это особенно важно для:
- кочек/ступенек с гладкими фронтами (в которых дорога имеет ненулевую скорость),
- энергетики (работа дороги = ∑ F_tire * z_road_dot),
- оптимизации (градиенты/гладкость при smooth_contacts).

Изменения внесены в:
- `pneumo_solver_ui/model_pneumo_v8_energy_audit_vacuum_patched_smooth_all.py`
- `pneumo_solver_ui/model_pneumo_v8_energy_audit_vacuum_patched.py`

### 2) Исправление демпфирования отбойников (freeze‑устойчивость)

**Было:** демпфирование отбойников завязывалось на `s_dot` из функции объёмов, который может быть искусственно обнулён на упоре штока.  
**Стало:** отбойники получают `delta_dot = zw_dot - z_body_dot` из кинематики.

Изменения внесены в:
- `pneumo_solver_ui/model_pneumo_v8_energy_audit_vacuum_patched_smooth_all.py`
- `pneumo_solver_ui/model_pneumo_v8_energy_audit_vacuum_patched.py`

### 3) Аналитическая `road_func_dot(t)` + fallback на численную производную

В `*_smooth_all.py` добавлена поддержка:
- `road_func(t) -> R^4` (как было)
- `road_func_dot(t) -> R^4` (новое)

Если `road_func_dot` не задана — используется центральная разность с `road_dot_eps`
(по умолчанию `1e-4`).

Также расширен генератор стандартных тестов `_compile_suite_test_inputs()`:
- для синус‑профилей и «кочек» добавляется `road_func_dot`.

### 4) Скорость/ускорение дороги в выходных данных

Добавлены выходные поля:
- `скорость_дороги_ЛП_м_с`, `...`
- `ускорение_дороги_ЛП_м_с2`, `...`

Это полезно для:
- верификации `pen_dot`,
- анализа динамики контакта,
- энерго‑аудита (работа дороги).

### 5) Механический энерго‑аудит в smooth‑модуле

В `*_smooth_all.py` добавлен механический баланс:

`ΔE_mech = W_road + W_axay + W_pdv(gauge) - D_tire - D_bump`

Логируются:
- энергии: `энергия_механика_Дж` и компоненты (KE, Ug, пружины, шины, отбойники)
- интегралы: `работа_дороги_Дж`, `работа_inertial_axay_Дж`, `работа_p_dV_gauge_Дж`
- потери: `потери_шины_Дж`, `потери_отбойники_Дж`
- ошибка: `ошибка_энергии_механика_Дж`, `ошибка_энергии_механика_отн`
- диагностический интеграл: `p_dV_abs_minus_gauge_Дж`

Итоговые значения добавлены в `df_atm` (одна строка), чтобы оптимизатор мог использовать эти метрики напрямую.

### 6) Унификация ключей параметров отбойников

В patched‑модуле добавлена совместимость:
- новые ключи: `bumpstop_k`, `bumpstop_c`
- старые ключи: `жёсткость_отбойника`, `демпфирование_отбойника`

В smooth‑модуле добавлены `k_stop_param`/`c_stop_param` с таким же fallback.

## Совместимость и влияние на результаты

- Исправления **изменят результаты** в сценариях, где дорога движется (ступенька/кочка/синус), потому что раньше демпфирование шины считало базу неподвижной.
- Исправления **изменят результаты** в сценариях с упорами/отбойниками, потому что раньше демпфирование могло «отключаться» из‑за freeze.
- Это ожидаемое и правильное изменение: прежнее поведение было физически некорректным.
