# Release Notes — R24

Дата: 2026-01-22

## Главное

### Анимация (2D/3D)

- Исправлена логика **`sync_playhead`** в 2D-компоненте `mech_anim`:
  - controls (Play/Pause/ползунок времени) больше не выключаются в режиме синхронизации;
  - controls пишут состояние в общий playhead (localStorage ключ `pneumo_play_state`), поэтому 2D/3D/таймлайн движутся синхронно;
  - если playhead ещё не создан (первый заход/чистый localStorage/смена набора данных), компонент создаёт корректное начальное состояние сам.
- Это убирает типичный «кажется, что всё пересчитывается бесконечно» при Play (особенно когда раньше анимация была в matplotlib fallback и Streamlit делал частые rerun).

### UI

- По умолчанию включён движок анимации **«Компоненты (SVG/Canvas)»** (fallback остаётся доступным вручную).

## Совместимость и риски

- Изменения касаются только фронтенда компонента `mech_anim` и выбора дефолтного движка.
- Матемодель/солвер не трогались.
