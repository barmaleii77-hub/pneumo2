Эта папка предназначена для "постоянных" данных пользователя,
которые должны сохраняться между перезапусками приложения.

По умолчанию сюда пишется файл:
  persistent_state/ui_state/autosave_profile.json

Он содержит:
- параметры модели (base)
- диапазоны оптимизации (ranges)
- набор тестов (suite)
- часть UI-состояния (выборы/настройки виджетов)

Путь можно переопределить переменной окружения:
  PNEUMO_STATE_DIR

Примечание:
WORKSPACE (PNEUMO_WORKSPACE_DIR) может быть сессионным (например, в runs/ui_sessions/...)
и используется для временных файлов, логов прогонов, кешей.
