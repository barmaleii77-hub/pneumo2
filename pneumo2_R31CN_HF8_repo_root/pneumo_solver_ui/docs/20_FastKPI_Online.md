# Быстрый онлайн‑KPI без pandas (v6.45)

Этот документ описывает *практический следующий шаг* после `compile_only` и `observe`:

- вместо `simulate()` → `df_main` → вычисление KPI по DataFrame,
- делаем **онлайн‑проход**: интегрирование RK2 + `observe(state, t)` и обновление KPI «на лету».

Это полезно для:
- массовой оптимизации параметров (тысячи прогонов),
- быстрых регрессий физики (инварианты, самопроверка),
- подготовки к градиентным методам (в дальнейшем).

## Что сделано

Добавлен инструмент:

- `pneumo_solver_ui/tools/fast_kpi_worldroad.py`

Он запускает world‑road v9 в режиме `compile_only=True` и печатает JSON с KPI.

## Как запустить

```bash
python -m pneumo_solver_ui.tools.fast_kpi_worldroad
```

Полезные опции:
- `--test-index N` — выбрать тест из `default_suite.json`.
- `--fully-smooth` — включить `fully_smooth_mode` (для сглаженных контактов/клапанов/потоков).
- `--json-out path` — сохранить KPI в JSON файл.

## Какие KPI считает

Сейчас инструмент считает минимальный, но полезный набор:
- `pR3_max_bar_abs` — максимум давления в ресивере 3.
- `roll_max_deg`, `pitch_max_deg` — максимумы |крен| и |тангаж|.
- `body_acc_z_rms_mps2` — RMS вертикального ускорения рамы (по RHS, без `np.gradient`).
- `lift_frac_any`, `lift_frac_wheels_mean` — отрыв колёс (доля шагов).
- `F_tire_min_N` — минимальная нормальная сила.
- `rod_margin_*` и `rod_speed_*` — запасы по штокам и максимальные скорости.
- `t_cross_Pmid_s` — время пересечения уставки `Pmid` после `t_step`.

Также печатаются диагностические поля:
- `rhs_t0_*` — ускорения RHS на t=0 (удобно для проверки нулевой позы).
- `dt`, `t_end`, `wheel_coord_mode`, `smooth`.

## Замечания по физике и корректности координат

1) KPI считаются из `observe()`, который использует **те же формулы**, что и основной цикл `simulate()`.
2) Контакт с дорогой вычисляется через **точку контакта** (`zw_contact`) и профили `z_road(t)`.
3) Нулевая поза определяется так же, как в основной модели: «ровная дорога», статическая подвеска,
   штоки на середине хода (по рассчитанному `spring_x0`), заданные начальные давления.
