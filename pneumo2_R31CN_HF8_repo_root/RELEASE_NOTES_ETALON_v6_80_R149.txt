RELEASE: ETALON_v6_80_R149
DATE: 2026-02-19

Ключевые изменения (по сравнению с R148):

1) Desktop Animator: ABS-координаты как закон
   - Анимация геометрии всегда берёт **абсолютные** каналы расчётной модели.
   - Никаких скрытых baseline/offset/"нулевой позы" со стороны Animator.
   - `*_rel0` допускаются только для графиков/анализа и используются только явно.

2) Жёсткая проверка контракта при загрузке NPZ
   - Если в NPZ отсутствуют обязательные ABS-каналы геометрии (дорога/колёса/рама по 4 углам),
     Animator теперь **останавливается с понятной ошибкой**, вместо молчаливой отрисовки нулей.

3) Документация и HELP
   - Обновлены docs по rel0: rel0 = удобство для анализа, но анимация = ABS.
