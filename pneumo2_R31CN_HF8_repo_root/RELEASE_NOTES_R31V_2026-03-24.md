# RELEASE NOTES — PneumoApp_v6_80_R176_R31V_2026-03-24

Дата: 2026-03-24

## Что исправлено

- Исправлено странное поведение поперечных полос дороги в Desktop Animator.
- Видимые cross-bars больше не строятся через привязку к ближайшим строкам текущего road mesh.
- Убран forced terminal cross-bar на дальнем краю viewport-а.

## Техническая суть

Раньше:
- world-anchored целевые позиции полос вычислялись корректно, но сами полосы брались по ближайшим строкам текущего `s_nodes` mesh-а;
- это давало мелкое дрожание/ступенчатость при сдвиге окна;
- дополнительно оставался один лишний бар, привязанный к последней видимой строке viewport-а.

Теперь:
- поперечные полосы строятся по точным `s_targets` в world frame;
- для каждой полосы геометрия берётся прямой интерполяцией текущих `x/y/z_left/z_center/z_right/normal`;
- forced edge bar отключён, так что в кадре остаются только реальные world-anchored полосы.
